Student Grades Calculator

This C++ application lets users efficiently input, calculate, and manage student grades. It supports dynamic handling of homework and exam scores using object-oriented programming principles, including the Rule of Three. Users can choose between average or median for final grade calculation, import student data from files, or generate random scores. The program outputs formatted tables sorted by name or surname, making it suitable for educational use and large datasets.

Features

Dynamic homework handling with std::vector

Final grade calculation by average or median

Random score generation or file import

Formatted output tables, sortable by name or surname

Full implementation of the Rule of Three in the Person class

Easily extendable and well-documented code

Final Grade Formula:
Final Grade = 0.4 × (Homework Average/Median) + 0.6 × Exam
